CALENDÁRIO

 


O QUE É:

Esta tela nos mostra um calendário com as tarefas agendadas por período.

COMO UTILIZAR:

Ano (1): Seleciona o ano do calendário mostrado a esquerda, não é possível retroceder o ano para antes do ano corrente, mas é possível avançar.

Mês (2): Seleciona o mês do calendário mostrado a esquerda, ao selecionar entre janeiro e dezembro os dias do mês são atualizados.

Dia (3): Seleciona o dia do mês e mostra os agendamentos para aquele dia.

Agendamento (4): Esta coluna mostra o nome do agendamento. Ao clicar no nome do agendamento o usuário é levado a uma tela de novo agendamento.

Cliente (5): Esta coluna mostra o nome do cliente.

Destinatário (6): Esta coluna mostra o destinatário.

Usuário (7): Esta coluna mostra o usuário que criou o agendamento.

Última execução (8): Esta coluna mostra a última vez que o agendamento foi executado.

Programação (9): Informa a programação do agendamento e a data de início.
