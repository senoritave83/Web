CADASTRO DE OPERADORES/ADMINISTRADORES

O QUE É:

A partir desta tela o usuário pode cadastrar ou editar os dados dos operadores/administradores do sistema Equipe OnLine.

COMO UTILIZAR:

Nome do Operador (1): Deverá ser colocado o nome do operador/administrador.

Caixa de seleção - Administrador (2): o usuário do sistema deverá determinar se o novo operador será um operador comum ou administrador do sistema. Se a caixa de seleção estiver selecionada (marcada), o novo operador será administrador do sistema, tendo acesso a todas as informações e configurações do sistema.

Login (3): Deverá ser colocado o login do usuário a ser utilizada pelo operador/administrador para acessar o Sistema Equipe OnLine.

Senha (4): Deverá ser colocada a senha a ser utilizada pelo operador/administrador para acessar o Sistema Equipe OnLine.

Após o término da edição dos campos acima descritos o usuário deverá acionar o botão Salvar (5) para confirmar o cadastro ou Voltar (6), para retornar à tela anterior.
