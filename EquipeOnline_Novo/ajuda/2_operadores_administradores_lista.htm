LISTA DE OPERADORES E ADMINISTRADORES

O QUE É:

A Lista de Operadores e Administradores é uma tela que apresenta todos os operadores/administradores que estão cadastrados no Sistema Equipe OnLine. A partir desta tela é possível acessar os dados, bem como editar ou adicionar novos dados.

COMO UTILIZAR:

Acessando dados de operadores/administradores cadastrados: Para acessar os dados de um operador/administrador cadastrado basta clicar em cima do Nome do Operador (1). Após clicar no nome o usuário é levado para a Tela de Cadastro de Operadores.

Caixa de Seleção (2): Seleciona o registro do operador/administrador.

Selecionar todos (3): Seleciona todos os registros

Excluindo Operador: Para excluir um ou mais operadores e todos os seus dados selecione as Caixas de Seleção (2) correspondentes e clique no botão Excluir (5).

Incluindo Operador: Para incluir um operador e todos os seus dados, clique no botão Adicionar (4).

Voltar para tela inicial de Configurações: Para voltar para a tela inicial de configurações, clique no botão Voltar (6).
