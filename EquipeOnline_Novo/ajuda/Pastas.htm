PASTAS


INTRODUÇÃO:

O sistema Equipe Online permite que o operador monte diversas Ordens de Serviço e envie posteriormente. Neste caso as Ordens de serviço estarão disponíveis no menu.


O QUE É:

Esta tela permite o acesso às Ordens de Serviço que estão armazenadas e são utilizadas para envio simultâneo de uma série de Ordens de Serviço


COMO UTILIZAR:

Ao lado de cada Ordem de serviço aparecerá uma Caixa de Seleção (1), que permite o Operador escolher, por rádio, quais ordens de Serviço poderão ser enviadas.

Após a seleção o operador deve clicar no botão Enviar (4).

Para enviar todas as mensagens armazenadas, o operador pode selecionar o botão Sel. Todas (2) e, em seguida o Enviar (4).

Após o envio, as Ordens de Serviço adquirem o status de "Enviada" e estarão disponíveis para consulta na Lista de Ordens de Serviço.

Para apagar uma O.S. armazenada selecione a Caixa de Seleção (1) correspondente a O.S. e Apagar (3).
