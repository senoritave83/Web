CADASTRO DE RESPOSTAS

PRIMEIRO NÍVEL DE RESPOSTAS


O QUE É:

Esta tela apresenta a lista de nove possíveis mensagens, configuradas de acordo com as necessidades da empresa, que o membro da Equipe de Campo terá que responder quando estiver executando uma determinada ordem de serviço.


COMO UTILIZAR:

Adicionando ou editando respostas padrão: Para adicionar ou alterar os dados das respostas o usuário deve clicar diretamente nas Caixas de Preenchimento (1), numeradas de 1 a 9 e digitar as respostas.

Adicionando cores: O usuário pode relacionar cores diferentes para cada resposta dada pelo responsável em campo. Para isto basta o usuário selecionar a cor correspondente na Caixa de Seleção de Cores (2) que fica ao lado de cada resposta.

Subníveis: Através da tela de cadastro de respostas é possível verificar se para cada resposta cadastrada existe ou não subníveis relacionados O ícone (4) representa a existência de subníveis. O ícone     (5) representa que não há subníveis relacionados à esta resposta.

Na opção alterar(7), é possível definir dados que serão requeridos do usuário remetente no momento do envio da resposta.

Apagando Respostas: Para apagar uma determinada resposta o usuário deve selecionar a Caixa de Seleção (6) correspondente e clicar no botão EXCLUIr (9).

Apagando Subníveis: Para apagar um subnível de resposta completo basta o usuário clicar no Ícone X (3).

Após a escolha das cores e o preenchimento das caixas de preenchimento, o usuário deve clicar no botão Salvar (8).
