EXPORTAÇÃO DE DADOS


O QUE É:

A ferramenta de Exportação de Dados disponível na aplicação Equipe Online, permite a exportação dos dados presentes na Tela de Histórico O sistema permite a exportação de dados no formato .txt e .xml, permitindo a utilização em outros sistemas ou softwares.


COMO UTILIZAR:

Exportar Dados para o formato .txt: Após aparecer a Tela de Exportação de Dados, o Operador deve selecionar na Caixa de Combinação (1) a opção Texto Delimitado. Após selecionar a opção o usuário deve escolher o separador que será utilizado para a exportação dos dados (ponto e vírgula, Tab ou outro determinado pelo usuário). A caixa de seleção Incluir Respostas (2) deve ser selecionada se o Operador quiser exportar todas as respostas gravadas pelos usuários no atendimento às Ordens de Serviço. Depois de feitas as seleções o usuário deve acionar o botão Exportar (3) para concluir a operação.

Exportar Dados para o formato .xml: Após aparecer a Tela de Exportação, o Operador deve selecionar na Caixa de Combinação (1) a opção XML. A caixa de seleção Incluir Respostas (2) deve ser selecionada se o Operador quiser exportar todas as respostas gravadas pelos usuários no atendimento às Ordens de Serviço. Depois de feitas as seleções o usuário deve acionar o botão EXPORTAR (3) para concluir a operação.
