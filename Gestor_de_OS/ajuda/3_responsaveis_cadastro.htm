CADASTRO DE R&Aatilde;DIOS


O QUE É:

A partir desta tela o usuário pode cadastrar ou editar os dados dos rádios dos usuários do Equipe Online.


COMO UTILIZAR:

Nome (1): Deverá ser colocado o nome do usuário do rádio.

Telefone (2): Deverá ser colocado o telefone do rádio no formato apresentado na tela.

ID (3): Deverá ser colocado o ID do aparelho Nextel do rádio no formato apresentado na tela.

Observação (4): Poderão ser colocadas observações a respeito do rádio. Esta informação não é de preenchimento obrigatório e tem o limite máximo de 2000 caracteres.

Caixa de Seleção - Grupos (5): Cada Grupo cadastrado no sistema está representado por uma caixa de seleção. Deverão ser selecionadas (marcadas) as caixas cujo grupo será incluído à um rádio que está sendo cadastrado.

O usuário deverá selecionar pelo menos um grupo para o rádio.

O usuário logado só visualiza os grupos de sistemas Nextel OnLine aos quais ele está cadastrado como Administrador.

Após o término do preenchimento/edição dos campos acima descritos o usuário deverá acionar o botão Salvar (6) para confirmar o cadastro ou Voltar (7), para retornar à tela anterior.
