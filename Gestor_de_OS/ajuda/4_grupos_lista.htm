LISTA DE GRUPOS


O QUE É:

Esta tela apresenta a lista de grupos que estão cadastrados no Sistema Equipe OnLine. A partir desta tela é possível acessar os dados bem como adicionar novos grupos.


COMO UTILIZAR:

Acessando dados de grupos cadastrados: Para acessar os dados de um grupo cadastrado basta clicar em cima do Nome do Grupo (1). Após clicar no nome o grupo é levado para a Tela de Cadastro de Grupos onde é possível realizar alterações como, adicionar ou excluir algum rádio pertencente ao grupo. Para apenas visualizar detalhes (7) sobre o grupo, na própria lista clique na seta (2) de detalhamento. Para omitir os detalhes de um grupo clique na seta correspondente (2).

Excluindo Grupo: Para excluir um ou mais grupos e todos os seus dados selecione as Caixas de Seleção (3) correspondentes e clique no botão Excluir (5).

Incluindo Grupo: Para incluir um grupo e todos os seus dados, clique no botão Adicionar (4).

Voltar para tela inicial de Configurações: Para voltar para a tela inicial de configurações, clique no botão Voltar (6).
