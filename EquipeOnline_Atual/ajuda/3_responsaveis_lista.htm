LISTA DE R&Aatilde;DIOS


O QUE É:

A Lista de Rádios é uma tela que apresenta os Rádios das Equipes de Campo que estão cadastrados nos Sistemas Nextel OnLine. A partir desta tela é possível acessar os dados bem como adicionar novos responsáveis.

 Esta tela é apresentada em forma de listagem e mostra o nome do rádio, grupos, status e a data de criação do rádio nos Sistemas Nextel OnLine.

COMO UTILIZAR:

Acessando dados de rádios cadastrados: Para acessar os dados de um rádio cadastrado basta clicar em cima do rádio (1). Após clicar será levado para a Tela de Cadastro de Rádios.

Excluindo Rádio: Para excluir um ou mais rádios e todos os seus dados selecione as Caixas de Seleção (2) correspondentes e clique no botão Excluir (4).

Incluindo Rádios: Para incluir um rádio clique no botão Adicionar (3).

Voltar para tela inicial de Configurações: Para voltar para a tela inicial de configurações, clique no botão Voltar (5).
