ALTERAR OPÇÕES DE RESPOSTA

 

O QUE É:


Nesta tela há opções de observações na resposta, é possível definir dados que serão requeridos no momento do envio da resposta.


COMO UTILIZAR:

preenchimento(1): neste campo o usuário poderá definir se a requisição de dados será opcional*, obrigatória ou se não será requerido nenhuma informação de quem responde. É possível também definir o formato**(2) da informação requerida.

Adicionando campos: O usuário pode também inserir mais campos de informações que achar necessário, podendo definir no campo texto(3) que tipo de informação será solicitada à quem responde. Após a definição dos campos adicionais, o usuário deve clicar no botão adicionar(4) para confirmar a inserção do mesmo. Ao lado de cada campo adicionado há o botão apagar(5), ao clicar nele o mesmo é apagado definitivamente.

Para finalizar e ativar os campos, o usuário deve clicar no botão salvar(6).

*Por padrão já está pré-definido o preenchimento opcional.
**Por padrão o formato alfanumérico já está pré-definido.
