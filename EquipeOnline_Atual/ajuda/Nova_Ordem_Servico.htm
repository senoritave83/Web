NOVA ORDEM DE SERVIÇO


O QUE É:

Para enviar uma nova ordem de serviço o Operador deverá acessar a Tela de Criação de Nova Ordem de Serviço.


COMO UTILIZAR:

Número da Ordem de serviço: Este número é um número seqüencial, atribuído pela aplicação, através de um formato escolhido pelo usuário na tela de configurações.

Cliente(1): Este campo permite identificar o nome do cliente cujo serviço é destinado. O Operador pode simplesmente digitar o nome do cliente ou, através do símbolo (2) utilizar a lista de clientes cadastrados na tela de configurações.

Endereço (3): Este campo permite a entrada dos dados do endereço do cliente. Este endereço poderá estar cadastrado no sistema e vir automaticamente quando o usuário selecionar o nome do cliente ou poderá ser digitado.

Rádio (4): Através desse campo o Operador escolhe qual o rádio da equipe de campo receberá a Ordem de Serviço. A seleção é feita através da caixa de combinação, que trará uma lista de todos os membros da Equipe de Campo que podem receber as ordens de serviço.

PRIORIDADE (5): é possível definir entre prioridade média, alta ou baixa para uma ordem de serviço.

Descrição do Serviço (6): Neste campo deve ser digitada a descrição do serviço a ser realizado. Podem ser utilizadas as MENSAGENS PADRÃO (7), escolha uma delas e clique em ADICIONAR(8).

Tempo Limite de Atendimento (9): Neste campo o Operador poderá cadastrar o Tempo Limite que o integrante da Equipe de Campo deverá cumprir a ordem de serviço.

Enviar (12): Envia a ordem de serviço para o rádio.

Agendar (11): Caso o Operador não queira enviar a ordem de serviço imediatamente após o preenchimento, ele poderá agendar o envio para outra data/horário. Verificar seção Agendar Ordem de Serviço.

Armazenar O.S. (10): A opção armazenar O.S. é utilizado nos casos em que o operador queira salvar a O.S., porém ainda não tem previsão de quando vai enviar. Desta forma, a O.S. permanecerá guardada e acessada através do menu "Pastas".

A função armazenar O.S. também pode ser utilizada quando o operador queira enviar as Ordens de Serviço em batelada para um determinado rádio. Desta forma ele pode montar inúmeras O.S. para um mesmo rádio e enviá-las de uma única vez no menu "Pastas".
