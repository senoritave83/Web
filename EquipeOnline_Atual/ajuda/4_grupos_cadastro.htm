CADASTRO DE GRUPOS


O QUE É:

A partir desta tela o usuário pode cadastrar ou editar os dados dos grupos cadastrados no sistema.


COMO UTILIZAR:

Nome do Grupo (1): Deverá ser colocado o nome do grupo.

Caixa de Seleção - Rádios (2): Cada Rádio está representado por uma caixa de seleção. Deverão ser selecionadas (marcadas) as caixas cujo respectivo rádio pertencerá ao grupo.

Após o término do preenchimento dos campos acima descritos o usuário deverá acionar o botão Salvar (3) para confirmar o cadastro ou Voltar (4), para retornar à tela anterior.
